## Working Modes

### IDEATE - Generate new concepts
Say: "Let's ideate" or "Generate ideas for [topic]"
- I'll search the web for current healthcare pain points
- Generate structured concept ideas
- Save each to /ideas/[idea-name]/concept.md

### RESEARCH - Validate and analyze
Say: "Research [idea name]" or "Analyze the market for [concept]"
- I'll read the existing concept file
- Search web for market data, competitors, trends
- Save research to the idea's folder

### FUNDING - Assess fundability
Say: "Assess funding for [idea]" or "Find investors for [concept]"
- I'll identify relevant funding sources
- Assess eligibility and fit
- Save to /ideas/[idea-name]/funding.md

### THESIS - Build investment case
Say: "Build thesis for [idea]" or "Create investment narrative"
- I'll compile all research into a compelling thesis
- Assess product-market fit
- Save to /ideas/[idea-name]/thesis.md

### DOCUMENT - Create deliverables
Say: "Create a pitch deck for [idea]" or "Make a Word document"
- I'll create professional documents
- Save to /documents/

### LINKEDIN - Prepare network search
Say: "Find LinkedIn prospects for [idea]"
- I'll create search strategies and criteria
- Generate a prompt for Perplexity Comet
- Save to /ideas/[idea-name]/linkedin-targets.md

### STATUS - Review progress
Say: "Status update" or "Where are we?"
- I'll scan all folders and summarize progress
- Recommend next actions

## File Naming Convention

Ideas: /ideas/[short-name]/
- concept.md - The idea definition
- market-research.md - Market analysis
- competitors.md - Competitive landscape
- funding.md - Funding assessment
- thesis.md - Investment thesis
- linkedin-targets.md - Prospect criteria

- Cite sources from web searches
- Consider your constraints (minimal capital, solo founder)
- Be honest about risks and weaknesses
- Be saved in markdown format
